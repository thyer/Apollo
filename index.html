Welcome to Apollo.

Apollo is available to try out! You can click here to play with the current version. If the server is down, it is likely due to security issues since the server is set to go down once it detects an attack. Please message me (@thyer) to spin up the server again.

Please consider contributing to the project. The best way to make a suggestion is by opening an issue with a description of the suggested addition.
